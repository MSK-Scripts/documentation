# v3.0.0

**Major release**

- Full rewrite of the whole library with new features and better performance.
- New framework and inventory bridge architecture (ESX, QBCore, ox_core and fully STANDALONE), with lazy loaded modules and a dual API where every function is available as `MSK.Function(...)` and `exports.msk_core:Function(...)`.
- New NUI built on React, Vite and TypeScript (Notify, Input, Numpad, Progressbar and TextUI), shipped prebuilt so the server never needs npm.

For the full API reference see the rest of this documentation.
