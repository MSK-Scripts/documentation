# v3.1.2

**Changed**

- **The NUI was rebuilt on updated dependencies.** The web interface now builds on React 19, Vite 8, TypeScript 7 and FontAwesome 7. There is no API or behavior change, the Lua side is untouched.

- **FontAwesome 7 ships only woff2.** The unused `.ttf` font files were removed from `web/dist`, so the built interface is a little smaller.

To update, replace `fxmanifest.lua` and `web/dist`. This is a pure NUI rebuild with no Lua or API change.
