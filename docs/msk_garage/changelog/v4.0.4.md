# v4.0.4

- Fixed that helicopter/aircraft garages showed no vehicles

- Fixed that one corrupt vehicle entry could empty the whole list
- Hardened park-in/park-out against vehicle/upgrade injection and type bypass
- Several UI fixes (rename dialog vs. Escape, double-open guard, marker opacity)
