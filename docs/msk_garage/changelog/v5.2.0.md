# v5.2.0

- Added luxu_admin support for the admin dashboard: staff groups from luxu_admin v2 (which are not FiveM ACE principals) are now recognized for dashboard access; configurable in config/static.lua via Config.LuxuAdmin (auto / on / off, optional group mapping and an on-duty requirement)
- Fixed heli/aircraft garages staying empty when the vehicle shop stored the type as 'air': 'air' is now accepted as a synonym for the helicopter and aircraft categories

## Changed files

```text
config/static.lua
server/admin/permissions.lua
shared/classes.lua
```
